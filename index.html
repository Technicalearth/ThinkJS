What is Think JS?

Think JS is a JavaScript library, which is fast, secure, and cross plaform and easy to use, you can try it in console.





Sample

Try it in the console.


// To select an element and making a click event

think("#btn").click(function(){
    alert("btn clicked.");
});


// To see the element

think("#btn");


// To see the text

think("#btn").text();


// To see the html

think("#btn").html();
